우리 커뮤니티는 프로젝트 개시를 위해 [dooboo-cli](https://github.com/dooboolab/dooboo-cli)를 활용합니다.
이 도구는 필요한 구성 요소들을 효율적으로 통합하여 프로젝트 시작을 용이하게 해주며, 7년 동안 꾸준히 유지보수되어 최신 스택을 잘 반영하고 있습니다.

## 프로젝트 만들기

먼저, Node.js가 정상적으로 설치된 터미널에서 아래 명령어를 입력합니다.
'개발환경' 섹션을 확인하고 지침을 따른 분들은 이미 필요한 환경이 준비된 상태일 것입니다.

```bash
npx dooboo init
```

<img src={ImgDoobooInit} alt="dooboo-init" />

이어서, 아래와 같은 방식으로 프로젝트명을 파스칼 케이스(PascalCase)로 입력하시면 프로젝트가 생성됩니다.

<img src={ImgDoobooInitRun1} alt="dooboo-init-run-1" />

dooboo-cli에 대한 자세한 내용은 [이 블로그 글](https://medium.com/crossplatformkorea/%EC%83%88%EB%A1%9C%EC%9A%B4-dooboo-cli-5c60e17a87e0)에서 확인하실 수 있습니다.
